# AssetFlow - Enterprise Asset & Resource Management System

## 📌 Overview
AssetFlow is a web-based application that helps organizations manage and track assets such as laptops, furniture, vehicles, and shared resources. It simplifies asset registration, allocation, and resource booking through a centralized platform.

## ✨ Features
- Secure Login & Authentication
- Dashboard with Asset Statistics
- Asset Registration & Management
- Asset Allocation
- Resource Booking
- Search & Filter Assets
- Role-Based Access

## 🛠️ Tech Stack
- **Frontend:** React.js
- **Backend:** Spring Boot
- **Database:** MySQL
- **Authentication:** JWT
- **Tools:** Git, GitHub, Maven

## 🚀 Getting Started

### Backend
```bash
cd backend
mvn spring-boot:run
```

### Frontend
```bash
cd frontend
npm install
npm start
```

## ⚙️ Database

Create the database:

```sql
CREATE DATABASE assetflow;
```

Update `application.properties` with your MySQL credentials before running the project.
